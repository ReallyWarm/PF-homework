# Sort Data in File
Write a program to sort lines of `IDs`, `names`, and `items` in the `data.txt` file by ID, then store them in a file named `output.txt`.

### data.txt
```
1 Hale Pen
10 Ben Watch
4 Peter Spider
8 Jim Banana
```

### output.txt
```
1 Hale Pen
4 Peter Spider
8 Jim Banana
10 Ben Watch
```
